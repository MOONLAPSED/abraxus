# Abraxus

## Usage; testing, fuzzing, linting, etc.

```bash
make install
make lint
make format
make test
make bench
make pre-commit-install
pdm run python main.py
```

## first run in a minit?
```
pipx upgrade pdm
pdm update
pdm cache clear
pdm install --clean
```
